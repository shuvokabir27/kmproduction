

# বাইরের কাজ (ফ্রিল্যান্স প্রজেক্ট) ম্যানেজমেন্ট — প্ল্যান

## সারসংক্ষেপ
সাইডবারে "বাইরের কাজ" নামে আলাদা মেনু আইটেম থাকবে। এখানে অন্যের শুটিংয়ে ক্যামেরার কাজের প্রজেক্ট তৈরি, টিম মেম্বার অ্যাসাইন, প্রজেক্ট অনুযায়ী আলাদা রেট সেট, আয়-ব্যয় হিসাব এবং মেম্বার পেমেন্ট ট্র্যাক করা যাবে।

## ডাটাবেস (২টি নতুন টেবিল)

### 1. `freelance_projects`
| কলাম | টাইপ | বিবরণ |
|---|---|---|
| id | uuid PK | |
| name | text | প্রজেক্টের নাম (যেমন: "রাজু নাটক শুটিং") |
| client_name | text | ক্লায়েন্টের নাম |
| client_phone | text (nullable) | ক্লায়েন্ট ফোন |
| project_date | date | কাজের তারিখ |
| location | text (nullable) | লোকেশন |
| total_budget | numeric default 0 | মোট বাজেট/চার্জ |
| total_expense | numeric default 0 | মোট খরচ |
| status | text default 'upcoming' | upcoming / ongoing / completed / paid |
| notes | text (nullable) | |
| created_by | uuid (nullable) | |
| created_at | timestamptz default now() | |

RLS: শুধু অ্যাডমিন CRUD

### 2. `freelance_assignments`
| কলাম | টাইপ | বিবরণ |
|---|---|---|
| id | uuid PK | |
| project_id | uuid | freelance_projects ref |
| member_id | uuid | profiles ref |
| role_label | text | ভূমিকা (ক্যামেরাম্যান, প্রডাকশন, এডিটর ইত্যাদি) |
| rate | numeric default 0 | এই প্রজেক্টে তার রেট |
| is_paid | boolean default false | পেমেন্ট হয়েছে কিনা |
| paid_amount | numeric default 0 | |
| notes | text (nullable) | |
| created_at | timestamptz default now() | |

RLS: অ্যাডমিন CRUD, মেম্বার নিজেরটা দেখতে পারবে

## ফ্রন্টেন্ড

### নতুন পেজ: `src/pages/AdminFreelance.tsx`
- **প্রজেক্ট লিস্ট**: কার্ড ভিউতে প্রজেক্টের নাম, ক্লায়েন্ট, তারিখ, স্ট্যাটাস, বাজেট, লাভ দেখাবে
- **প্রজেক্ট তৈরি/এডিট ডায়ালগ**: নাম, ক্লায়েন্ট, তারিখ, লোকেশন, বাজেট
- **টিম অ্যাসাইন**: প্রজেক্ট কার্ডে ক্লিক করলে মেম্বার যুক্ত করা যাবে, প্রতিটি মেম্বারের ভূমিকা ও রেট সেট করা যাবে
- **পেমেন্ট ট্র্যাকিং**: প্রতিটি মেম্বারের পেমেন্ট স্ট্যাটাস (পেইড/আনপেইড) ম্যানেজ
- **সামারি ভিউ**: মাসিক আয়, খরচ, লাভ — ড্যাশবোর্ড স্টাইলে

### ট্যাব স্ট্রাকচার:
1. **প্রজেক্ট সমূহ** — সব প্রজেক্ট কার্ড
2. **মাসিক হিসাব** — মাস অনুযায়ী আয়/ব্যয়/লাভ সামারি

### সাইডবার আপডেট (`AppSidebar.tsx`)
- টিম ম্যানেজমেন্ট সেকশনে "বাইরের কাজ" মেনু আইটেম যুক্ত হবে (Briefcase আইকন)

### রাউট (`App.tsx`)
- `/admin/freelance` → `AdminFreelance`

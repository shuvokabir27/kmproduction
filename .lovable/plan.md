
# ফ্রিল্যান্স প্রজেক্ট — ক্লায়েন্ট ভিউ ও শুটিং লাইনআপ

## সারসংক্ষেপ
বাইরের ক্লায়েন্ট একটি পাবলিক লিংকের মাধ্যমে তার প্রজেক্টের আর্থিক তথ্য ও শুটিং লাইনআপ দেখতে পারবে। অ্যাডমিন সিন-বাই-সিন লাইনআপ তৈরি করবে এবং সেটি PDF হিসেবে ডাউনলোড করা যাবে।

## ডাটাবেস পরিবর্তন

### 1. `freelance_projects` টেবিলে নতুন কলাম
- `share_token` (text, unique) — ক্লায়েন্টের জন্য পাবলিক শেয়ার লিংক তৈরিতে ব্যবহার হবে

### 2. নতুন টেবিল: `freelance_scenes`
| কলাম | টাইপ | বিবরণ |
|---|---|---|
| id | uuid | Primary Key |
| project_id | uuid | FK → freelance_projects |
| scene_number | integer | সিন নম্বর |
| description | text | সিনের বিবরণ |
| location | text | শুটিং লোকেশন |
| characters | text | অভিনেতা/চরিত্র |
| sort_order | integer | ক্রম |
| created_at | timestamp | তৈরির সময় |

### 3. RLS
- অ্যাডমিন সবকিছু ম্যানেজ করতে পারবে
- পাবলিক ইউজার `share_token` দিয়ে শুধু নিজের প্রজেক্টের scenes দেখতে পারবে

## ফ্রন্টএন্ড পরিবর্তন

### 1. `AdminFreelance.tsx` — লাইনআপ ম্যানেজমেন্ট
- প্রতিটি প্রজেক্টে "লাইনআপ" বাটন যুক্ত
- সিন যোগ/এডিট/ডিলিট করার ফর্ম
- "শেয়ার লিংক কপি করুন" বাটন — ক্লায়েন্টের জন্য পাবলিক লিংক তৈরি

### 2. নতুন পেজ: `FreelanceClientView.tsx`
- পাবলিক পেজ (লগইন লাগবে না)
- URL: `/project/:token`
- দেখাবে:
  - প্রজেক্টের নাম, তারিখ, লোকেশন
  - বাজেট ও পেমেন্ট সামারি (মোট বাজেট, মোট খরচ, লাভ)
  - টিম মেম্বার তালিকা (নাম, ভূমিকা, রেট, পেমেন্ট স্ট্যাটাস)
  - সিন-বাই-সিন শুটিং লাইনআপ

### 3. PDF ডাউনলোড
- ক্লায়েন্ট ভিউ থেকে "PDF ডাউনলোড" বাটন
- PDF-এ থাকবে:
  - শুটিং লাইনআপ টেবিল (সিন নম্বর, বিবরণ, লোকেশন, চরিত্র)
  - আর্থিক সামারি (বাজেট, খরচ, টিম খরচ)

## রাউট
- `/project/:token` — পাবলিক ক্লায়েন্ট ভিউ

## ফাইল পরিবর্তন:
1. ডাটাবেস মাইগ্রেশন (freelance_scenes টেবিল + share_token কলাম + RLS)
2. `src/pages/AdminFreelance.tsx` — লাইনআপ ও শেয়ার লিংক
3. `src/pages/FreelanceClientView.tsx` — নতুন পাবলিক পেজ
4. `src/App.tsx` — রাউট যুক্ত
